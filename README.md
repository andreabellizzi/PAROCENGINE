# PAROCENGINE
PAROC stays for Par O Cazz Engine, a long and maybe never ending project, with high effort initially and then long period of drop because of demotivation, of creating a game without a generic engine but with low customizable, highly game dependent game engine.

# Philosophy
There exist general purpose game engine, why create a game without it? Because using a game engine implies learning all stuff of the game engine and not knowning how it works internally, and I hate to open a tool with thousand of options that I had no clue to what they did, what they are useful for etc. I want only what I need, stop having more of the necessary.

AI will shift the thinking from process of creating to just create, YTYH approach, You Think You Have, (I like acronyms), and we'll end up to have a lot of stuff, maybe also beutiful, but we do not know where they came from. Only if we know where things came from we can have control of it. Let's use the AI when we already know how to do stuff and simply want speed up things.
